# Генератор фирменной графики

Браузерное веб-приложение для самостоятельного создания визуальных материалов в фирменном стиле.

---

## Требования

Для сборки приложения на компьютере должны быть установлены:

- **Node.js** версии 18 или выше — [nodejs.org](https://nodejs.org)
- **npm** (устанавливается вместе с Node.js)

---

## Сборка

1. Распакуйте архив с исходным кодом в любую папку.

2. Откройте терминал (командную строку) и перейдите в эту папку:
   ```
   cd путь/к/папке
   ```

3. Установите зависимости:
   ```
   npm install
   ```

4. Запустите сборку:
   ```
   npm run build
   ```

5. После завершения сборки в папке появится директория `dist/`. Это и есть готовое приложение.

---

## Размещение

Папку `dist/` нужно разместить на любом веб-хостинге, поддерживающем раздачу статических файлов.

**Варианты размещения:**

- **Собственный сервер** — скопируйте содержимое `dist/` в корневую директорию вашего веб-сервера (например, `/var/www/html` для Nginx или Apache).
- **Netlify** — перетащите папку `dist/` на страницу [app.netlify.com/drop](https://app.netlify.com/drop). Приложение будет доступно по сгенерированному адресу немедленно, без регистрации.
- **Vercel** — загрузите папку `dist/` через интерфейс [vercel.com](https://vercel.com).

> Приложение не требует базы данных, серверной логики или дополнительных настроек. Достаточно любого хостинга со статическими файлами.

---

## Локальный запуск (для проверки)

Если хотите запустить приложение локально, не размещая его на хостинге:

```
npm run preview
```

После запуска откройте браузер и перейдите по адресу `http://localhost:4173`.
